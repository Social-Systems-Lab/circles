# Kamooni (circles)

Kamooni is an open-source community platform for connection, collaboration, coordination, and collective action.

## What this repo contains
- Next.js / TypeScript application
- MongoDB-backed app data
- Mongo-native chat and notifications
- MinIO-based object storage
- Qdrant for vector / semantic features
- Docker-based local and production workflows

## Current status
This repo no longer uses Matrix for the main chat/DM system. Any older Matrix references should be treated as legacy unless explicitly marked otherwise.

## Fast start
1. Clone the repo
2. Copy the environment file
3. Start Docker services
4. Run the app locally
5. Open the app in your browser
