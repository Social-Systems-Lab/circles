## Genesis2 deploy

From `/root/circles/circles` on Genesis2:

```bash
./circles/deploy-genesis2.sh main
```

Optional branch argument:

```bash
./circles/deploy-genesis2.sh codex/refactor/chat-sidebar-search
```
